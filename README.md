# 澳门大学 CIS 博士资格考试资源

用于准备 澳门大学（UM）计算机与信息科学系（CIS）博士资格考试（QE）的相关资源。

## 📘 关于本仓库

本仓库收集并整理与澳门大学 CIS 博士资格考试相关的学习资料，包括：

* 主题总结
* 学习笔记
* 练习题
* 参考资料
* 个人理解

所有内容**仅用于学术分享与学习**。  
使用这些资料时，请尊重原作者以及学校的相关政策。

## 📚 内容概览

* 核心 QE 科目与大纲
* 来自课程或线上资料的笔记
* 潜在考试主题
* 练习题集
* 用于深入学习的补充参考资料

## 🧭 整理说明

博士资格考试对于许多同学来说都是一个重要但压力较大的阶段。  
建立本仓库的初衷，是希望把分散的资料、笔记和经验系统地整理起来，帮助后来的同学更高效地准备考试，减少重复摸索的成本。

如果未来这一考试制度能够被进一步优化，甚至有一天不再需要通过这种形式来评估博士生的研究能力，那当然再好不过。  
在此之前，希望这些资料至少能够为正在准备 QE 的同学提供一些实际帮助。

## 🙏 致谢

本仓库中的资料来源于多年来许多学生、校友和同伴的努力与贡献。  
本仓库旨在延续这一传统，使未来的学生能够继续完善、扩展并传承这些知识。
